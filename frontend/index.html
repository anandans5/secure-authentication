Secure Authentication System

A secure phone authentication project built with modern backend technologies and industry-standard security practices.

Project Overview

This project demonstrates a secure phone number authentication workflow using Express.js, SQLite, Twilio SMS and JWT. It follows a modular backend architecture with secure API design and authentication best practices.

Security Features

  • Phone OTP Authentication
  • JWT Authentication
  • OTP Expiration
  • OTP Attempt Protection
  • Rate Limiting
  • Helmet Security Headers
