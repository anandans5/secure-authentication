Secure Authentication System

Secure phone number authentication using OTP, JWT and a modular Express backend.

Overview

This project demonstrates a complete authentication workflow using Express.js, SQLite, Twilio SMS and JSON Web Tokens. It follows a modular backend architecture with multiple security controls to protect against common authentication attacks.
